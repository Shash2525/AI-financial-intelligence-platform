https://ai-financial-intelligence-platform-orpin.vercel.app

# ⚙ Installation

## Clone repo

```bash
git clone https://github.com/Shash2525/AI-financial-intelligence-platform.git
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

---

# 🔑 Environment Variables

Create `.env` inside backend folder:

```env
GROQ_API_KEY=your_key
```
